# 单例模式（Singleton）

## 简介

单例模式是一种创建型设计模式，确保一个类只有一个实例，并提供全局访问点。它常用于需要控制资源访问的场景，如数据库连接、线程池等。

## UML 类图

````mermaid
classDiagram
    class Singleton {
        - instance: Singleton
        + getInstance(): Singleton
    }
````

## 实现方式

1. **懒汉式**（线程不安全）
2. **懒汉式**（线程安全）
   - 使用 `synchronized` 关键字确保线程安全，但会影响性能。
   - 使用双重检查锁定（Double-Checked Locking）来提高性能。
